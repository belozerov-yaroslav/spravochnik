Для более полной подготовки к экзамену рекомендуем дополнительно изучить данные материалы:

  • Сайт ФГБНУ "ФИПИ" ссылка

    • На сайте можно скачать демоверсии, спецификации, кодификаторы. Можно воспользоваться открытым банком заданий ЕГЭ.

  • Решу ЕГЭ ссылка

    • Образовательный портал для подготовки к экзаменам, на котором можно тренироваться в выполнении отдельных заданий, распределённых по группам. Есть возможность проходить сгенерированный тестовые варианты.

  • kpolyakov.spb.ru ссылка

    • Сайт одного из авторов учебника информатики для средней школы (10–11 классы). На сайте есть рекомендации по методам решения каждой конкретной задачи. Есть ьренажёр компьютерного ЕГЭ.
