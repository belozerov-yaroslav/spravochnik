Справочник создавался, как продукт в школьном проекте.

Цель справочника: помочь ученикам в сдаче ЕГЭ по информатике.

Цель достигается с помощью ознакомления учеников со структурой заданий экзамена, и порядком его проведения.

Для успешного выполнении заданий ученикам надо практиковаться. Для тренировки сущестуют разделы "Вариант ЕГЭ" и "Задания".

Краткое описание разделов сайта:

  • Вариант ЕГЭ

    • При открытии раздела пользователю предлагается решить сгенерированный набор заданий, который примерно повторяет структуру заданий ЕГЭ по информатике. Тренирующийся читает условие задания и вводит свой ответ в соответствующее поле под заданием. В конце выполнения заданий пользователь нажимает на кнопку "Отправить на проверку", и система сообщает пользователю сколько первичных баллов он набрал.

  • Задания

    • Предоставляет доступ ко всем заданиям, доступным в базе данных справочника. Задания распределены по своему номеру в структуре ЕГЭ. Также можно найти задание по уникальному ID, который всегда указывается в заголовке задания.

  • Структура заданий

    • Содержит информацию о структуре заданий экзамена.

  • Справка

    • Содержит информацию о порядке проведения экзамена.

  • Полезные ссылки

    • Ссылки на ресурсы, которые также могут помочь подготовиться к экзамену.
